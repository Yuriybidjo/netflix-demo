# My Calendar Layout

A modern calendar dashboard built with HTML, CSS, and JavaScript.

## 🚀 Features

- **Live Date:** Updates automatically every hour.
- **Ordinal Suffixes:** Displays dates as 1st, 2nd, 8th, etc.
- **Custom Fonts:** Uses Inter and KyivTypeTitling.
